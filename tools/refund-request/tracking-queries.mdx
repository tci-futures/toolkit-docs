## Overview

The "All Queries" section provides real-time visibility into your refund requests, allowing you to track progress and stay aligned with the accounts team throughout the process.

<Info>
	All refund queries you've submitted are displayed in one centralised location
	for easy monitoring.
</Info>

## Accessing your queries

<Steps>
<Step title="Navigate to All Queries">
  Click on "All Queries" in the main navigation menu of the Refund Request App.
  
  <Frame>
  <img src="/images/refund-request-all-queries.png" alt="All Queries navigation button in the main menu" />
  </Frame>
</Step>

<Step title="View your refund requests">
  You'll see a list of all your submitted refund queries with key information displayed.
</Step>
</Steps>

## Understanding query status

Your refund requests will display one of the following statuses:

<AccordionGroup>
<Accordion title="Submitted" icon="paper-plane">
  Your request has been successfully submitted and is awaiting review by the accounts team.
  
  <Note>
  Queries under £2,000 move directly to processing. Higher-value queries await approval.
  </Note>
</Accordion>

<Accordion title='Pending approval' icon='clock'>
	Your request requires approval from your manager (£2,000+) or compliance
	(£10,000+) before the accounts team can process it.
	<Tip>
		If your query has been pending for an extended period, you may want to
		follow up with your manager or compliance team.
	</Tip>
</Accordion>

<Accordion title='Approved' icon='check-circle'>
	All required approvals have been granted, and the accounts team is now
	processing your refund request.
</Accordion>

<Accordion title='In progress' icon='spinner'>
	The accounts team is actively working on processing your refund.
</Accordion>

<Accordion title='Completed' icon='circle-check'>
	The refund has been processed successfully and the transaction is complete.
	<Check>You should see the refund reflected in the client's account.</Check>
</Accordion>

<Accordion title="Rejected" icon="circle-xmark">
  Your refund request has been rejected. Check the query details for the reason and any next steps.
  
  <Warning>
  Review the rejection reason carefully. You may need to submit a new request with corrected information.
  </Warning>
</Accordion>
</AccordionGroup>

## Query information

Each refund query displays the following details:

| Information           | Description                               |
| --------------------- | ----------------------------------------- |
| Query reference       | Unique identifier for your refund request |
| Transaction reference | The transaction being refunded            |
| Client name           | Name of the client receiving the refund   |
| Refund amount         | Total amount being refunded               |
| Status                | Current stage in the refund process       |
| Submission date       | When you submitted the request            |
| Last updated          | Most recent activity on the query         |

<Tip>
	Click on any query to view full details, including notes, approval history,
	and communication with the accounts team.
</Tip>

## Viewing query details

<Steps>
<Step title="Select a query">
  Click on any refund query from your list to open the detailed view.
</Step>

<Step title='Review comprehensive information'>
	The detailed view includes: - Complete submission details - Current status and
	history - Approval timeline (if applicable) - Notes and comments from the
	accounts team - Acturis reference number
	<Frame>
		<img

			alt='Detailed view of a refund query showing all information'

	</Frame>
</Step>

<Step title="Add notes or updates">
  If you need to provide additional information, use the notes section to communicate with the accounts team.
</Step>
</Steps>

## Filtering and searching

Find specific queries quickly using the search and filter options:

<Tabs>
<Tab title="Search">
  Use the search bar to find queries by:
  - Client name
  - Transaction reference
  - Query reference number
  
  <Tip>
  Enter partial terms to see matching results as you type.
  </Tip>
</Tab>

<Tab title='Filter by status'>
	Use status filters to view: - All queries - Pending approval only - In
	progress only - Completed queries - Rejected queries This helps you focus on
	queries requiring your attention.
</Tab>

<Tab title="Sort">
  Sort your queries by:
  - Submission date (newest/oldest)
  - Refund amount (highest/lowest)
  - Last updated
  - Client name (A-Z)
</Tab>
</Tabs>

## Email notifications

You'll receive email updates when:

- Your refund request is submitted (with Acturis reference)
- Approval is required from your manager
- Your request is approved or rejected
- The accounts team completes processing

<Note>
	Check your email regularly to stay informed of any status changes or requests
	for additional information.
</Note>
